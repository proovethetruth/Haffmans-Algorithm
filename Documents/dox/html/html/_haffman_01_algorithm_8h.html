Функция для генерации дерева Хаффмана

Создаётся приоритетная очередь, которая в последствии заполняется узлами (символ-частота). Затем создаётся новый узел двоичного дерева, частота которого равна сумме частот двух символов с наивысшей частотой, которые затем становятся его листьями и удаляются из приоритетной очереди. Процесс повторяется до тех пор, пока все вершины не станут листьями дерева

Parameters
[in]freqСписок пар символ-частота
Returns
Указатель на корень бинарного дерева Хаффмана

Функция для генерации строковой репрезентации бинарного дерева Хаффмана

Для генерации строковой репрезентации дерева используется прямой обход (NLR). Если текущий узел является листом (не имеет потомков) , то в строку добавляется единица и символ этого узла. Если текущий узел не является листом, в строку записывается ноль и происходит рекурсивный вызов функции

Parameters
[in]nodeУказатель на корень дерева
[in]resultСсылка на строку, в которой будет сохранён результат

Функция для записи байта в бинарный файл

Эта функция упаковывает восемь битов в байт, затем записывает его в бинарный файл. Последний байт может оказаться незаполненным, ввиду чего использование этой функции требует последующего вызова функции FlushBits() для вставки дополнительных нулей
