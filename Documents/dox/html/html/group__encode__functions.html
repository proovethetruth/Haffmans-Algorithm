◆ build_tree()

Node * build_tree ( std::unordered_map< char, int >  freq)

Функция для генерации дерева Хаффмана

Создаётся приоритетная очередь, которая в последствии заполняется узлами (символ-частота). Затем создаётся новый узел двоичного дерева, частота которого равна сумме частот двух символов с наивысшей частотой, которые затем становятся его листьями и удаляются из приоритетной очереди. Процесс повторяется до тех пор, пока все вершины не станут листьями дерева

Parameters
[in]freqСписок пар символ-частота
Returns
Указатель на корень бинарного дерева Хаффмана

Функция для определения кодов символов

Рекурсивный прямой обход дерева (NLR). При переходе к левой ветви, на следующий уровень рекурсии копируется строка с добавленной к ней нулём, к правой - с единицей. Если текущий узел является листом (не имеет потомков)

  • сопоставить символу этого узла полученную в ходе рекурсии строку.
    Parameters
    [in]rootУказатель на корень дерева
    [in]strСтрока, необходимая для работы рекурсии
    [in]huffmanCodeПустой список пар символ-строка для записи кодов

Функция для архивации текста

Алгоритм Хаффмана кодирует наиболее часто встречаемые символы бинарными кодами меньшей длины, а редко встречаемые символы бинарными кодами большей длины. Таким образом, заменяя символы на их соответствующие коды, полученные в ходе обхода дерева Хаффмана можно сжать файл
