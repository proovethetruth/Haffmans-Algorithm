Функция для считывания байтов текста из бинарного файла

В заархивированном файле первый символ означает кол-во дополнительных нулей вставленных функцией Flush_Bits(). Следующее за ним число - длина строковой репрезентации дерева. Затем следует разделительный символ, непосредственно строковая репрезентация дерева, и сжатый текст.

Parameters
[in]infileСсылка на файл, открытый в двоичном режиме
[in]textСсылка на строку, в которую будет сохранён результат
[in]tree_sizeДлина строковой репрезентации дерева
[in]zerosКол-во дополнительных нулей

void writeBinaryTree ( Node *  node,
std::string &  result 
)

Функция для генерации строковой репрезентации бинарного дерева Хаффмана

Для генерации строковой репрезентации дерева используется прямой обход (NLR). Если текущий узел является листом (не имеет потомков) , то в строку добавляется единица и символ этого узла. Если текущий узел не является листом, в строку записывается ноль и происходит рекурсивный вызов функции

Parameters
[in]nodeУказатель на корень дерева
[in]resultСсылка на строку, в которой будет сохранён результат
